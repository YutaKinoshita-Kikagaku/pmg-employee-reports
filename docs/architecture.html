行動・メンタル
Slack/Notion から抽出した行動宣言、メンタル状況、エルダー/メンター観察

4. レポート作成フロー（毎週金曜）

  1. PCS CSV を Google Drive から読み込み。個別採点シートから点数・コメントを抽出
  2. Slack の育成連携スレッド・メンバー相談スレッドから当週の投稿を取得（MCP 経由）
  3. Notion から定例議事録・行動宣言の更新を取得（MCP 経由）
  4. 既存 HTML に差分を反映（4箇所チェックリスト: サマリ / 点数推移 / 採点詳細 / 範囲別）
  5. デフォルト表示タブを当該週に切り替え
  6. ローカルで表示確認
  7. git commit + push → GitHub Pages に自動デプロイ
更新チェックリスト（4箇所必須）

① サマリタブ: 折れ線グラフ JS データ + ステータステーブル
② 点数推移テーブル: 項目別週平均推移の該当週セル
③ 採点詳細タブ: score-bar + 項目テーブル（点数/Good/More）
④ 範囲別スコアタブ: 該当週の範囲・回数・色分け
